# Webstack 网址导航

### 项目介绍

本项目是一个 Typecho 网址导航主题，基于 [Webstack 钻芒二开版](https://www.zmki.cn/5366.html) 。

原项目为 [WebStackPage.github.io](https://github.com/WebStackPage/WebStackPage.github.io)，基于 bootstrap 前端框架开发，各后台及主题版本由广大网友提供。
